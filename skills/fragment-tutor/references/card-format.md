# 卡片格式 · Card Format

Backs Stage 2–4 of fragment-tutor.

## 1. 一卡一念原则

切卡判据:把这张卡单独发给一个没看过原文的人,他能完整理解并答出自测题 → 合格。
做不到就说明这张卡装了超过一个观点,继续拆。

常见切分错误:
- 按原文章节切(章节常含多个观点)→ 应按观点切
- 把背景铺垫单独成卡(没有自测点)→ 背景压缩进相关观点卡的"核心"里
- 例子单独成卡 → 例子永远依附于观点

## 2. 四要素写法

**核心(≤120 字)**: 第一句就是结论;术语首次出现必须当场用大白话解释;
禁止"我们将在后文看到"式悬置。

**例子**: 优先级:用户行业的真例 > 生活类比 > 原文自带例子。类比要标注
失效边界("这个类比在 X 情况下不成立"),防止学错。

**自测题**: 必须是回忆式(answer from memory),不是判断题。
- ✓ "用自己的话说,X 和 Y 的区别是什么?"
- ✓ "如果遇到 [场景],按这张卡你会先做什么?"
- ✗ "X 是不是比 Y 好?"(可猜)
- ✗ "本卡讲了什么?"(太泛)

**钩子**: 二选一——预告下一卡的未解问题,或一个可以在通勤路上空想的应用题。

## 3. 难度标定

按 Stage 1 的学习目的调整:

| 目的 | 核心深度 | 自测题型 |
|------|----------|----------|
| 考试 | 贴考纲表述,保留必要术语 | 接近真题句式 |
| 工作应用 | 只讲怎么用 + 什么时候不能用 | 场景应用题 |
| 兴趣了解 | 讲清直觉,可砍推导 | 复述题 |

## 4. 复习时刻表规则

固定四次:当天睡前 / D+1 / D+7 / D+30。每次只做自测题:

- 流畅答出 → 该卡毕业,不再出现
- 答出一半 → 重读"核心",留到下一次
- 完全没答上 → 重读整卡,该卡的间隔重置(明天再来)

抽查模式(Stage 4)打分口径:对 = 要点齐;部分对 = 缺要点或术语含糊;
没答上 = 空白或方向错。打分必须诚实,"差不多对"按部分对处理。
