# 钩子模式库 · Hook Patterns

Backs Stage 2–3 of copy-title-forge.

## 1. 十大钩子模式

每个模式 = 触发的心理 + 模板 + 示例 + 翻车点。

| 模式 | 心理 | 模板 | 翻车点 |
|------|------|------|--------|
| 数字盘点 | 降低阅读成本 | N 个 X,第 K 个最 Y | N>9 显得注水 |
| 反差 | 打破预期 | [低起点],我却 [高结果] | 结果必须真实可证 |
| 悬念缺口 | 好奇心 | 我 [做了 X] 之后,才知道 [被隐藏的 Y] | 正文必须回答 Y |
| 利益直给 | 省时间 | X 分钟学会 [具体技能],附 [可拿走的东西] | 附的东西必须真附 |
| 身份点名 | 相关性 | [具体人群] 都该看看的 X | 人群越窄越准 |
| 错误清单 | 损失回避 | 别再 [常见做法] 了,这是 [代价] | 不可夸大代价 |
| 亲历叙事 | 真实感 | 我 [时间] 内 [做了什么],记录全过程 | 必须真是亲历 |
| 对比评测 | 决策辅助 | X 和 Y 用了一个月,差距在 [维度] | 结论要有依据 |
| 反常识 | 认知冲突 | 关于 X,你可能一直做反了 | 正文论证要扎实 |
| 时效借势 | 紧迫相关 | [近期事件] 之后,X 的玩法变了 | 事件须真实相关 |

## 2. 平台调参表

| 平台 | 长度 | 风格 | 关键点 |
|------|------|------|--------|
| 小红书 | ≤20 字 | 口语 + 1–2 个 emoji,第一人称 | 关键词前置(搜索流量),封面文案与标题分工 |
| 公众号 | ≤24 字 | 克制,可用破折号分层 | 标题 + 摘要是一组,摘要补利益 |
| 抖音/快手 | 前 3 秒口播钩子 | 一句话,口语化 | 标题是文案,钩子在开头台词 |
| 知乎 | 问题式或判断式 | 理性,不用 emoji | 蹭已有高热问题 > 自拟标题 |
| B站 | ≤25 字 | 可玩梗,信息量要足 | 标题与封面字互补不重复 |

## 3. 评分细则 (每维 0–2 分,共 10)

- **具体性**: 有具体数字/场景/人群 2 分;泛泛 0 分
- **利益清晰度**: 读者一眼知道能带走什么 2 分
- **好奇缺口**: 有想点开的未知项 2 分;全剧透 0 分;只有悬念没有信息也 0 分
- **可兑现度**: 正文能完全兑现 2 分;兑现不了直接判 0 并剔除
- **平台适配**: 符合调参表 2 分

≥8 分进 Top 3 候选;可兑现度为 0 的标题无论总分多少一律剔除并说明原因。

## 4. 快速诊断(用户拿旧标题来问"为什么没人点")

按顺序检查:有没有具体名词 → 利益在不在前半句 → 是否撞了平台连发三天的同款
句式 → 封面/开头是否与标题打架。逐条给出改法,再按 Stage 2 重锻 10 个。
